# NeuroLetter: Розпізнавання образів за допомогою багатошарового перцептрона (MLP)

Проєкт реалізує штучну нейронну мережу прямого поширення (Feedforward Neural Network) з нуля за допомогою бібліотеки
NumPy. Мережа призначена для розпізнавання матричних зображень п'яти літер алфавіту (а, б, с, д, е), навіть за наявності
спотворень та шуму.

## Постановка задачі

Розробити штучну нейронну мережу для класифікації п'яти символів, поданих у вигляді матриць розміром 6x6 пікселів.
Реалізувати алгоритм навчання мережі та провести дослідження впливу різних параметрів (кількості прихованих шарів,
кількості нейронів, функцій активації та алгоритмів навчання) на ефективність розпізнавання та швидкість збіжності.

## Аналіз завдання

+ **Вхідні дані**: Зображення літери 6x6 пікселів, розгорнуте в одновимірний масив (вектор розмірністю 36).
+ **Вихідні дані**: 3-бітний бінарний код, що ідентифікує літеру (наприклад, `0 0 0` для "а", `0 1 1` для "д").
  Розмірність вихідного шару – 3 нейрони.
+ **Тип задачі**: Багатомножинна бінарна класифікація (Multi-label classification). Оскільки сума цільових ймовірностей
  не обов'язково дорівнює 1, використання функції Softmax є недоцільним. Замість неї на вихідному шарі застосовується
  функція Sigmoid.
+ **Навчання**: Здійснюється методом зворотного поширення помилки (Backpropagation) із застосуванням різних алгоритмів
  оптимізації.

## Теоретичні відомості

Проєкт базується на класичній архітектурі [*MLP (Multi-Layer
Perceptron)*](https://www.geeksforgeeks.org/deep-learning/multi-layer-perceptron-learning-in-tensorflow/). Для навчання
реалізовано наступні компоненти:

1. **Функції активації**:
    - `Sigmoid`: Перетворює значення в діапазон (0, 1). Використовується на вихідному шарі для отримання ймовірностей.
    - `ReLU`: Повертає $0$ для від'ємних значень та $x$ для додатних. Запобігає проблемі згасаючого градієнта.
    - `Tanh`: Гіперболічний тангенс, перетворює значення в діапазон (-1, 1).
2. **Алгоритми оптимізації**:
    - `SGD` (Stochastic Gradient Descent): Класичний градієнтний спуск.
    - `RMSProp`: Адаптивний алгоритм, що запобігає згасанню кроку навчання завдяки використанню ковзного середнього для
      квадратів градієнтів.
    - `Adam` (Adaptive Moment Estimation): Найсучасніший алгоритм, що поєднує адаптивний крок (як у RMSProp) та
      інерцію (Momentum).
3. **Функції похибки**:
    * `MSE` (Mean Squared Error) – середньоквадратична похибка.
    * `BCE` (Binary Cross-Entropy) – бінарна крос-ентропія, ідеальна для бінарної класифікації.

## Короткий опис коду

Проєкт побудовано за принципами ООП та розділено на три основні модулі:

+ `utils.py` – відповідає за роботу з даними. Містить функції `load_data` (парсинг файлів `.train`) та
  `generate_noisy_dataset` (аугментація даних шляхом генерації зашумлених версій оригінальних матриць).
+ `models.py` містить абстрактний базовий клас `BaseNeuralNetwork`, де реалізовано усю математику: пряме поширення (
  `forward`), зворотне поширення (`backward`), розрахунок градієнтів та оптимізатори (SGD, RMSProp, Adam). Включає
  механізм *Early Stopping* (ранньої зупинки навчання). Від цього класу успадковуються конкретні мережі (
  `SigmoidNetwork`, `ReLUNetwork`, `TanhNetwork`).
+ `main.py` – точка входу. Скрипт завантажує дані та послідовно запускає 5 тестів з різними конфігураціями
  гіперпараметрів:
    1. Класичний перцептрон без прихованого шару (SGD).
    2. Мережа з 5 нейронами (ReLU + Adam).
    3. Широка мережа на 15 нейронів (Sigmoid + RMSProp).
    4. Широка мережа на 30 нейронів (Sigmoid + SGD).
    5. Глибока нейромережа з двома прихованими шарами [20, 10] (Tanh + Adam).
